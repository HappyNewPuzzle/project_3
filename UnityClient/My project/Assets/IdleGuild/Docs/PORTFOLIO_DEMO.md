# Idle Guild 포트폴리오 데모 가이드

## 2분 시연 순서

1. 자동 전투와 `NEXT BOSS` 진행 표시를 보여 줍니다.
2. 성장 메뉴에서 공격력과 공격속도를 강화합니다.
3. 스킬 메뉴에서 STAR BURST, SWIFT STRIKE, GUARDIAN LIGHT를 사용합니다.
4. 일곱 번째 전투의 보스 체력바, 제한시간, 공격 패턴을 보여 줍니다.
5. 장비 드롭과 자동 장착, 전투력 상승을 확인합니다.
6. Stage 4 전후의 동굴, Stage 7 전후의 설원 전환을 보여 줍니다.
7. 개발자 패널을 켜 실제 API 로그인과 서버 상태 복원 로그를 보여 줍니다.

## 설명할 기술 포인트

- Unity Coroutine과 Animator를 나눠 전투 흐름과 프레임 애니메이션을 관리했습니다.
- Sprite Sheet마다 다른 Pivot과 투명 여백을 캐릭터별 전투선으로 보정했습니다.
- 스킬 이펙트의 충돌 프레임과 서버/클라이언트 피해 표시 타이밍을 동기화했습니다.
- UI 패널은 한 번에 하나만 열어 모바일 화면 겹침을 방지했습니다.
- 서버는 JWT 사용자 식별, PostgreSQL 저장, 멱등 키, 낙관적 동시성을 담당합니다.
- Unity는 Mock API와 실제 API를 같은 인터페이스로 교체합니다.

- PostgreSQL 18 및 ASP.NET Core API Docker 컨테이너 healthy
- `20260716225618_AddSelectedHero` 마이그레이션 적용
- 신규 게스트 생성 성공
- 공격 4, 속도 2, 치명타 3 저장·복원
- 장비 Tier 2/3개, 지역 1, 스킬 2/2/2 저장·복원
- 선택 영웅 `black_cat` 저장·복원
- API 컨테이너 재시작 후 동일 JWT로 상태 조회 성공

## 포트폴리오 범위

학습용 MVP이므로 결제, 광고, 소셜 로그인, PvP, 라이브 운영 도구는 제외했습니다. 대신 자동 전투의 시각적 피드백과 서버 권위 저장 구조를 끝까지 연결하는 데 집중했습니다.
