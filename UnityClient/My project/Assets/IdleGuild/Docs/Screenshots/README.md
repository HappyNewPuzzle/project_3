# 포트폴리오 스크린샷

실제 Android 기기 또는 Unity Game View에서 아래 이름으로 캡처합니다.

- `01-auto-battle-hud.png`: 자동 전투 전체 HUD
- `02-hero-selection.png`: 소녀·검은 고양이 영웅 선택
- `03-regular-monster-combat.png`: 일반 몬스터 등장과 공격
- `04-boss-battle.png`: 보스 체력바·제한시간·공격
- `05-active-skills.png`: 액티브 스킬 이펙트
- `06-growth-equipment.png`: 성장·장비·전투력
- `07-server-restore.png`: 서버 저장·재접속 복원

권장 크기는 Android 세로 화면 원본 해상도이며, 개인정보·JWT·비밀번호는 캡처에 포함하지 않습니다.
